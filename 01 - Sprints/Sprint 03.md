## 🚀 Artefatos Esperados

1. **Design de UX/UI:** User Flow, Wireframes e link do Protótipo Interativo (Figma ou similar).
2. **Identidade Visual:** Manual simplificado da marca (Cores, Fontes e Logotipo).
3. **Engenharia de Software:** Diagrama de Classes UML.
4. **Banco de Dados:** Diagrama ER, Esquema Relacional e Scripts SQL (DDL para criação e DML para manipulação).
5. **Qualidade:** Definição de métricas baseadas na ISO 25010 relevantes para o projeto.
6. **Gestão:** Quadro de tarefas atualizado no GitHub Projects refletindo o progresso do grupo.
