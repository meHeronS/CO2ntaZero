# Front-end do Projeto

Este diretório referencia o código-fonte da interface do usuário (frontend) da aplicação **CO2ntaZero**.

## Tecnologias Utilizadas

*   **React.js**: Biblioteca JavaScript para construção de interfaces de usuário.
*   **CSS Modules**: Estilização modularizada.
*   **Axios**: Cliente HTTP para comunicação com o backend.

## Localização do Código

O código fonte encontra-se neste diretório.

## Como Executar

1.  No terminal, dentro desta pasta (`src/front`):
    ```bash
    npm install
    npm start
    ```
