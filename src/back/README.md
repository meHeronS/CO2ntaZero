# Back-end do Projeto

Este diretório referencia o código-fonte do servidor (backend) e da API da aplicação **CO2ntaZero**.

## Tecnologias Utilizadas

*   **Node.js**: Ambiente de execução JavaScript.
*   **Express.js**: Framework web para construção da API.
*   **Mongoose**: ODM para modelagem de dados no MongoDB.
*   **JWT**: Autenticação via JSON Web Tokens.

## Localização do Código

O código fonte encontra-se neste diretório.

## Como Executar

1.  No terminal, dentro desta pasta (`src/back`):
    ```bash
    npm install
    # Configure o arquivo .env
    npm run dev
    ```
